# COVID-19 Data Analysis Project

## 📊 Overview

This project analyzes COVID-19 data for India, United States, and Brazil to identify trends, patterns, and insights using Python and Power BI.

## 🛠 Tools & Technologies

* Python (Pandas, Matplotlib)
* Power BI
* Google Colab

## 📈 Key Analysis

* Total cases trend over time
* Daily new cases comparison
* Death rate analysis
* Country-wise comparison

## 📊 Dashboard

Interactive dashboard created using Power BI to visualize COVID-19 trends.

## 🔍 Insights

* United States had the highest total cases
* India showed rapid growth during peak periods
* Brazil showed fluctuating trends
* Death rate decreased over time

## 📁 Files Included

* Data cleaning & analysis notebook
* Clean dataset
* Power BI dashboard file
* Dashboard screenshot

## 🚀 Conclusion

This project demonstrates data cleaning, analysis, visualization, and storytelling skills using real-world data.
